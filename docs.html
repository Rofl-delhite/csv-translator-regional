Documentation

What it does?

Convert csv from English to different regional languages (Hindi, Marathi,Punjabi,Telugu). To do this, it make use of Google Translate API.

How to use it?

Prequesites
  • Node get it from Here.
  • Git get it from Here.
  • VsCode get it from Here .
Get the local copy of the repository
  • Create a folder and open in VsCode.
  • Use the follwing git commands :-
    1. git init Initializes empty git repository.
    2. git remote add origin {paste the github link of the respository} Now the remote to the respository is set to "origin".
    3. npm init Initializes node in this repository.
    4. git pull origin master This brings all code to your local machine.
    5. npm install This installs all the dependencies required for the project.
Run the project
  • node index.js Start the project.
  • Open the link http://localhost:3000 in the browser.
